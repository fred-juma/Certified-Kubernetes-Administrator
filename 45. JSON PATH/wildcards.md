##### 1 / 9 Wildcard - Dictionary
Develop a JSON path query to extract the expected output from the Source Data.


**Source Data:**

```json

{
  "property1": "value1",
  "property2": "value2"
}

```

**Expected Output:**
```json
[
  "value1",
  "value2"
]
```

**JSON Path query:**

```bash
$.*
```
